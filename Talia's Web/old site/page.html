How To Make A Website

Steps:

  1. Install Visual Studio Code
  2. Make a folder
  3. Open that folder in Visual Studio Code
  4. Make a new file in your folder and name the file ___.html
  5. Click on extensions and search "open in browser".
  6. Install "open in browser"
  7. Click with two fingers on the file and click "Open in Default Browser"
  8. Go to explorer and type in the code to the right
